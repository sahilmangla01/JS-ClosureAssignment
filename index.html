Q1.	"    function counter(){
            var counter = 0;
        
            function IncreaseCounter() {
                return counter += 1;
            };
        
            return IncreaseCounter;
        }
        
        var counter = counter();
        alert(counter());
        alert(counter());
        alert(counter());
        alert(counter());"

Output: 

            1
            2
            3
            4

Explanation: In the above code, the outer function return the reference of the inner function IncreaseCounter.IncreaseCounter increases the outer variable counter by 1. After that we have stored the return value of Counter into a variable counter. Now, We have called the inner function multiple times which will increase the counter by 1 each time. 

Q2.	"let count = 0;
            (function () {
              if (count === 0) {
                let count = 1;
                console.log(count); // What is logged?
              }
              console.log(count); // What is logged?
              })();"

Output: 

                1
                0

Explanation: In the above code, we give count value 0 with let keyword in global scope. when function run inside the block scope of if we redeclared count as 1 using let. As we know let is a block scope. so the first console will print the value 1 and second console is outside the block of if so it will give 0 as output. 

Q3.	"for (var i = 0; i < 3; i++) {

                3

Q. 7 
	"var a = 10;
            var x = (function () {
              var a = 12;
              return function () {
                alert(a);
              };
            })();
            x();"

Output:      12

Explanation: In the above code, First we have created a variable a and given it value (10). Then we have created a First-Order Function with the name x. inside this function we have redeclared and reassigned the value of our global variable to (12). And then we have created an inner function to get the output of a this inner function is the return value of our outer function. Finally we have called our function using x(); to get the output 12.
